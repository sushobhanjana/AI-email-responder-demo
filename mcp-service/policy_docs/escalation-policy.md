# Escalation & Urgency Policy

## Escalation Indicators
An email is considered an "Escalation" if it contains language indicating dissatisfaction, anger, or a request for higher-level intervention.

**Keywords & Phrases:**
- "unacceptable"
- "disappointed"
- "frustrated"
- "escalate this"
- "speak to your manager"
- "missed deadline"
- "breach of contract"
- "legal action"
- "ignoring my emails"

**Tone Analysis:**
- Aggressive or demanding tone.
- Excessive use of capitalization (shouting).
- Multiple question marks or exclamation points (e.g., "???", "!!!").

## Urgency Indicators
An email is "Urgent" if it requires immediate action within the same business day.

**Keywords:**
- "ASAP"
- "Urgent"
- "Immediately"
- "Today"
- "Blocker"
- "Critical"
- "System down"
- "Production issue"

## Classification Rules
1. If Escalation detected -> Category "Escalation", Priority "High", Action "Notify Manager".
2. If Urgency detected -> Priority "High".
